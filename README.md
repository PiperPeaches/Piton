# Piton | System Monitor
Piton is a simple system monitor writen in rust with soal perpose off having multiple tools in 1 app
Piton is usd to monitor how Cpu, Ram, Mounted Disks and Network Upload and Download 

## Getting started
To start using the app download it from the release and  run it and everything should work

## Support
> Piton Is made on linux for linux
If you want to run piton on a non Linux system you will need to build from source
